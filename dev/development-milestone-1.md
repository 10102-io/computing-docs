# Development Milestone 1

The first phase of the project focuses on the inheritance will type, building directly on the Safe wallet and infrastructure. We leave the security of the assets to a battle-tested, and popular wallet.&#x20;

Our module is an added layer to existing multi-sig wallets specialized in adding beneficiaries in flexible and convenient ways, offering the best of both worlds to web3 users.

Our smart contract repository is intentionally public, so that the financial operations that are sensitive don't require to blindly trust our code. We encourage reviews, independent research, and feedback.
