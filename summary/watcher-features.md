# Watcher Features

## Introduction

Authorized Watchers\
Limited visibility - Full visibility

While the watcher features can be used in various ways, we provide below a few real world use cases.

## Proxy designation

When designating a proxy as trustee, we give a specific Ethereum account the capability to connect and view some information about the legacy plans that have included it in the related proxy field. The shared information with the proxy would be limited to the account identifier (see below), and the beneficiaries. The public keys of the owner within the legacy plans are not fully displayed.

## Linking of multiple wallets

Single owner with multiple wallets is typical, and while it feels fragmented for a reason (security), creating a single view of one's wealth via legacy is the best form of consolidation anyone can get: preserved security, consolidated in future.
